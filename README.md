# Python Tic Tac Toe

A simple Tic Tac Toe game written in Python as part of my learning and practice.

## Project Structure
tictactoe-py/
 ├── tictactoe.py
 └── run_tictactoe.bat

## How to Run
1. Make sure Python 3.13 or newer is installed.
2. Run the game:
   - Double-click run_tictactoe.bat on Windows
   - OR run from the command line:
     py -3.13 tictactoe.py

## About This Project
This project demonstrates basic Python programming and game logic.
It will be improved over time as I continue learning Python.
