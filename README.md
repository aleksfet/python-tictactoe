# Python Tic Tac Toe

A simple GUI-based Tic Tac Toe game built using Python and Pygame as part of my learning and practice.

## Project Structure
tictactoe-py/
 ├── tictactoe.py
 └── run_tictactoe.bat

## How to Run
1. Make sure Python 3.13 or newer is installed.
2. Install dependencies (if needed):
pip install pygame

3. Run the game:
   - Double-click run_tictactoe.bat on Windows
   - OR run from the command line:
     python tictactoe.py

## Features
- Interactive GUI built with Pygame
- Turn-based gameplay (Player X and Player O)
- Win detection (rows, columns, diagonals)
- Tie detection
- Restart button to start a new game
- Close button to exit the game
- Improved win display (Player X / Player O)

## About This Project
This project demonstrates Python programming concepts including game logic, event-driven programming, and state management.
It has been iteratively improved with additional features and usability enhancements as I continue learning Python.
